# vue-todolist

## Frontend

https://xu-li-min.github.io/vue-todolist/#/

## Description

- 新增項目
- 完成項目(點擊要完成項目之文字)
- 編輯項目(雙擊要編輯項目之文字，編輯完成後 Enter 即可修改，Ese 則可取消編輯)
- 將代辦事項儲存至LocalStorage，並在載入時讀取
